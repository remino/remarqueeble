A tiny custom element tribute to the cursed glory of<marquee>, exposed as<re-marquee> and<re-marquee-ble>.

Install it with npm install remarqueeble, then importremarqueeble/auto or calldefineRemarqueebleElements() yourself.

Default

<re-marquee>

Default marquee: left, scroll, scrollamount 6, scrolldelay 85. Nisi nisi anim enim consequat pariatur reprehenderit. Proident cillum incididunt in laborum est duis quis.

Directions

direction="left"

← I crawl left into the neon ditch.

direction="right"

I flee right with suspicious confidence. →

direction="up"

↑ Upward scroll. Ancient guestbook energy rising from the crypt.

direction="down"

Downward scroll. A banner descending from the GeoCities ceiling. ↓

Scroll Speed

scrollamount="1"

Very slow. The dial-up modem is thinking.

scrollamount="12"

Fast. Too fast. The web page has escaped containment.

scrolldelay="200"

Slow delay. Chunky little steps across the screen.

Behaviour

behavior="scroll"

Scroll behaviour loops from one side to the other.

behavior="alternate"

I bounce between the walls like a screensaver DVD logo.

behavior="slide" loop="2"

Slides twice.

Presentation

bgcolor="var(--dress-bg-2)" width="18rem"

The background color and width come from marquee attributes.

hspace="2rem" vspace="1rem"

Horizontal and vertical spacing map to margins.

height="6rem" direction="up"

The explicit height overrides the vertical default of 200px.

API

Controlled with start() and stop().
